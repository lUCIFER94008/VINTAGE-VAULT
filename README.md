# 🛍️ VINTAGE.VAULT

VINTAGE.VAULT is a modern full-stack e-commerce platform focused on surplus and thrifted fashion. Built with a premium UI and scalable backend, it delivers a seamless shopping experience with admin-controlled product management and WhatsApp-based ordering.

---

## 🚀 Features

- 🧥 Surplus & Thrifted Clothing Store
- 🌑 Premium Dark UI with Gold Accent Design
- 🔐 Admin Authentication (Protected Dashboard)
- 📦 Product Management (Add Products with Images)
- ☁️ Cloudinary Image Upload Integration
- 🗄️ MongoDB Database (Mongoose)
- 📱 Fully Responsive (Mobile + Desktop)
- 💬 WhatsApp Order Integration (No payment gateway required)
- 🔄 Exchange / Replacement Support UI

---

## 🛠️ Tech Stack

- **Frontend:** Next.js (App Router), Tailwind CSS  
- **Backend:** Next.js API Routes  
- **Database:** MongoDB (Mongoose)  
- **Image Hosting:** Cloudinary  
- **Authentication:** Local Auth (Admin Protected Routes)
